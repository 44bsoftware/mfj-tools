# mfj-tools

Tax calculators for married couples filing jointly (MFJ) — and deciding whether to file separately (MFS).

Live site: https://mfjtools.com

## Stack

- Astro (static output)
- Tailwind CSS
- Hosted on Cloudflare Pages

## Local development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Output goes to `dist/`. Cloudflare Pages auto-deploys from the main branch.
